```
Install passbolt like you would normally using the installation scripts.
Load the data in the database using  mysql -u your_db_user -p your_db_name < backup.sql
Copy the server keys in config/gpg directory and load them in the keyring using gpg import (as described in the installation documentation). Something like: sudo su -s /bin/bash -c "gpg --home=/keyring/location --import /var/www/passbolt/config/gpg/serverkey_private.asc " www-data with --home matching the keyring location defined in passbolt.php if itâ€™s not the default.
Copy your passbolt.php in config directory
Copy the avatars in webroot/img/public/Avatar
```
